CSS Font Cheetsheet!

Property Purpose Example
font-family: This sets the font type. You can also set backup fonts seperated by a comma font-family: Helvetica, Gerogia, serif;
background-color: Sets the background color of the element. background-color: CadetBlue;
color: This sets the color of the font You can use preset CSS colors, hexadecimal, rgb, or hsl! color: #ABCDEF;
text-align: This aligns your text within an element. You can set left, right, or center. text-align: center;
font-weight: Sets the 'weight' or boldness of your text. font-weight: bold;
@import *FAV* Used to import things, such as fonts directly from a source (ie: googs) @import url('https://fonts.googleapis.com/css2?family=Reenie+Beanie&display=swap');

HTML Cheetsheet!

Tag Use Example
<link> Used to link various things such as a css stylesheet <link rel="stylesheet" href="style.css" type="text/css">
<head> This is where you put all your setup and metadata <head> link header metadata etc </head>

<img> Add images to your website! No closing tag <img src="google.com/img/neverendingpastabowlmukbang.gross>
